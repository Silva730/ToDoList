# To Do List — Frontend (Visual)

Este projeto é uma **To Do List apenas para visualização**, criada exclusivamente para compor meu portfólio e demonstrar habilidades em **frontend**.

## Objetivo

Exibir a construção de uma interface de lista de tarefas, com foco em layout, organização visual e experiência do usuário, **sem funcionalidade real de persistência ou lógica completa**.

## Características

- Interface estática para demonstração visual
- Design simples e organizado
- Componentes visuais de uma lista de tarefas
- Estrutura pensada para apresentação em portfólio

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Observações

- Projeto **não funcional**
- Nenhum dado é salvo ou processado
- Finalidade exclusivamente visual e demonstrativa

## Status do projeto

- Tipo: Frontend estático
- Uso: Portfólio
- Estado: Visualização apenas
